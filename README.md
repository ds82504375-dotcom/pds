# Prontuário de Sistema de lembretes em python

## Descrição
O projeto foi desenvolvido para se tornar um prontuário na palma da mão, um sistema que serve para registrar todas as informações do assistido. Utilizando a linguagem python e conceitos básicos da programação.

## Funcionalidades
- Exibe lembretes de cadastrados 
- Adicionar e remover tarefas
- Adicionar nome, horários e turnos a serem realizados
- Relatório do dia por turno

## Linguagens utilizadas
- Python
- Github

## Estruturas utilizadas
- Listas
- Dicionários
- Estrutura de decisão
- Estrutura de repetição

## Considerações finais
O Github possibilitou a organização e a evolução do sistema, permitindo o registro de cada etapa a serem seguidos e realizados.
